## Install

需要安装以下的环境：

* [node.js](https://nodejs.org/en/download/)
* [hexo](https://hexo.io/docs/index.html)

**Linux:**

```bash
# nodejs & npm
sudo apt-get install nodejs
sudo apt-get install npm
# hexo
npm install -g hexo-cli
```

**Windows:**
自行下载安装nodejs（已集成npm）

安装hexo：

```bash
npm install -g hexo-cli
```

## Local run

```bash
hexo g & hexo s
```

## Deploy

**Linux:**

```bash
deploy.sh
```

**Windows:**

```bash
deploy.bat
```
